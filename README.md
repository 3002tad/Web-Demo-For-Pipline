# Web_CNM Demo Website

Day la website demo don gian de test quy trinh pipeline.

## Cau truc thu muc

- `index.html`: Trang chinh
- `server.js`: Dau vao server local
- `assets/css/styles.css`: CSS
- `assets/js/app.js`: Logic phia client
- `data/`: Du lieu mau (categories, products, suppliers)
- `Input/tracking.js`: Dau vao script tracking
- `output/tracking-events.jsonl`: Su kien output mau

## Cach dung

1. Mo `index.html` tren trinh duyet, hoac
2. Chay server local (neu pipeline yeu cau).

## Ghi chu

- Du an chi dung de test pipeline.
- Cac file du lieu chi mang tinh demo.
